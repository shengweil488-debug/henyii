# 教师仪表板 (Teacher Dashboard) 使用说明

## 功能概述

教师仪表板 (`dashboard_teacher.php`) 是专门为教师用户设计的活动管理界面，提供以下核心功能：

### 🎯 主要功能
1. **查看全部活动** - 浏览系统中的所有活动
2. **创建新活动** - 快速创建新的活动
3. **活动搜索和筛选** - 按标题、地点、级别、状态进行筛选
4. **活动统计** - 实时显示活动数量统计
5. **多语言支持** - 支持中文、英文、马来文

### 📊 统计信息
- 总活动数
- 即将开始的活动数
- 进行中的活动数
- 已完成的活动数

### 🔍 筛选功能
- **搜索框** - 按活动标题或地点搜索
- **级别筛选** - School, District, State, National, International
- **状态筛选** - 即将开始、进行中、已结束

## 访问方式

### 1. 直接访问
```
http://localhost/henyii/public/dashboard_teacher.php
```

### 2. 通过登录系统
1. 访问登录页面：`http://localhost/henyii/public/login.php`
2. 使用教师账号登录
3. 系统会自动跳转到教师仪表板

## 权限控制

- 只有角色为 `teacher` 的用户才能访问此页面
- 其他角色用户会被重定向到主仪表板

## 界面特色

### 🎨 设计特点
- **现代化UI** - 使用渐变色彩和卡片式设计
- **响应式布局** - 支持桌面和移动设备
- **悬停效果** - 卡片悬停时有动画效果
- **图标丰富** - 使用 Font Awesome 图标

### 📱 移动端适配
- 自适应网格布局

## 技术实现

### 后端技术
- **PHP 7.4+** - 服务器端逻辑
- **MySQL** - 数据存储
- **PDO** - 数据库连接
- **Session** - 用户会话管理

### 前端技术
- **Bootstrap 5** - UI框架
- **Font Awesome 6** - 图标库
- **CSS3** - 自定义样式
- **JavaScript** - 交互效果

## 使用步骤

### 1. 查看活动
1. 页面加载后会自动显示所有活动
2. 使用筛选器查找特定活动
3. 点击"查看"按钮查看活动详情

### 2. 创建活动
1. 点击页面右上角的"创建活动"按钮
2. 或点击活动列表上方的"创建活动"按钮
3. 跳转到活动创建页面

### 3. 搜索和筛选
1. 在搜索框中输入关键词
2. 选择级别筛选
3. 选择状态筛选
4. 点击"搜索"按钮

## 故障排除

### 常见问题

1. **页面无法访问**
   - 检查文件权限
   - 确认PHP环境正常
   - 验证数据库连接

2. **显示空白页面**
   - 检查PHP错误日志
   - 确认数据库表存在
   - 验证用户会话状态

3. **样式显示异常**
   - 检查CSS文件路径
   - 确认Bootstrap CDN可访问
   - 验证Font Awesome加载

### 调试模式
在 `dashboard_teacher.php` 开头添加：
```php
error_reporting(E_ALL);
ini_set('display_errors', 1);
```
